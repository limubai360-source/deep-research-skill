# OpenClaw Instructions

Use this repository as a portable research skill pack.

## Entry Point

Primary workflow:

```text
skills/deep-research/SKILL.md
```

## When To Use

Use the skill for:

- deep research
- technical research
- market research
- competitive analysis
- policy / standards research
- literature review
- business analysis
- formal reports
- HTML research reports
- PPT-style HTML slide decks

## Compatibility Contract

The skill is plain Markdown and does not require a proprietary runtime. Follow `SKILL.md` as the source of truth, and load files in `references/` only when the task requires them.

If the user asks for a final external-facing report, remove construction artifacts and user misconception labels. Use corrected professional framing.
